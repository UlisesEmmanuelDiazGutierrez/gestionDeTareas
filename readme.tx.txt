Manual de Usuario: Sistema de Gestión de Tareas
1. Introducción
Bienvenido al Sistema de Gestión de Tareas. Este programa te permite organizar y
administrar tus tareas de manera eficiente. A continuación, encontrarás las instrucciones
detalladas para utilizar todas las funciones disponibles.
2. Requisitos del Sistema
• Este programa está diseñado para ser ejecutado en sistemas operativos Windows,
Linux o macOS.
• Asegúrate de tener un compilador de C instalado en tu sistema para poder compilar
y ejecutar el código fuente.
3. Menú Principal
El programa presenta un menú principal con las siguientes opciones:
1. Agregar nueva tarea: Permite agregar una nueva tarea a la lista.
2. Marcar una tarea como completada: Permite marcar como completada una tarea
existente.
3. Listar todas las tareas pendientes: Muestra una lista de todas las tareas que están
pendientes.
4. Listar todas las tareas: Muestra una lista de todas las tareas, tanto completadas como
pendientes.
5. Buscar por título: Permite buscar una tarea por su título.
6. Salir: Cierra el programa.
4. Instrucciones de Uso

A continuación, se detallan las instrucciones para utilizar cada función:
4.1. Agregar nueva tarea
Selecciona la opción 1 en el menú principal. Ingresa el título de la nueva tarea cuando se te
solicite. La tarea se agregará automáticamente a la lista.
4.2. Marcar una tarea como completada
Selecciona la opción 2 en el menú principal. Ingresa el número de la tarea que deseas
marcar como completada. Sigue las instrucciones adicionales que aparecerán en pantalla
para confirmar tu elección.
4.3. Listar todas las tareas pendientes
Selecciona la opción 3 en el menú principal. Se mostrará una lista de todas las tareas
pendientes.
4.4. Listar todas las tareas
Selecciona la opción 4 en el menú principal. Se mostrará una lista de todas las tareas, tanto
pendientes como completadas.
4.5. Buscar por título
Selecciona la opción 5 en el menú principal. Ingresa el título de la tarea que deseas buscar.
El sistema mostrará la tarea correspondiente si existe.
4.6. Salir del programa
Selecciona la opción 6 en el menú principal para salir del programa.
5. Notas Importantes
• Asegúrate de presionar "Enter" después de cada acción para continuar.
• En caso de ingresar datos incorrectos, el programa te proporcionará un mensaje de
error.

¡Gracias por utilizar el Sistema de Gestión de Tareas! Si tienes alguna pregunta o
encuentras algún problema, no dudes en consultar la documentación o ponerte en contacto
con el soporte técnico.
